#Jekyll
[jekyll home](https://jekyllrb.com)

##Quick Start

```
gem install jekyll bundler
jekyll new my-awesome-site
cd my-awesome-site
bundle exec jekyll serve
```
To view:

```
http://localhost:4000
```
